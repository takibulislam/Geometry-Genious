What is event bubble?

Event Bubbling is a concept in the DOM. It happens when an element receives an event, and that event bubbles up (propagated) to its parent and ancestor elements in the DOM tree until it gets to the root element.
we can access multiple events jusing event bubble, we we have to add event listener to the parent / anchestors and set target to those childs.
